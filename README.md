## 💻 Instalación 💻
- Ubicarse en la carpeta que contendrá el proyecto
- Abrir terminal de comandos
  - git clone https://github.com/elliotgaramendi/portfolio-eg.git
  - cd portfolio-eg
  - Abrir el proyecto con un servidor en un cliente
